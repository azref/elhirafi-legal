This Policy explains how your data is collected, used, and protected when you use the Al-Sanea Al-Hirafi mobile application. By using the Application, you agree to this Policy.

4. Data Collected

4.1 Data provided by the user

  • Account information (full name, phone number, email address).
  • Professional profile (profession, primary city, notification cities).
  • User-generated content (work requests, text and audio messages in the built-in chat).

4.2 Authentication

We use secure authentication systems. We may receive basic profile data depending on the login method used.

4.3 Photos and Media

The application only collects the profile photo chosen by the user and audio messages sent via the built-in chat to facilitate communication between artisans and clients. The application does not collect or store any images of products or goods.

4.4 Location Data (Manual Selection)

The application does not use GPS or your device's location services under any circumstances. Location data (primary city and notification cities) are manually selected by the user from a predefined list built into the application. This information is used exclusively for the smart filtering system to connect clients and artisans in the same region.

4.5 Automatically collected data

We may collect:

  • Crash reports.
  • Device information (model, operating system, application version).
  • Technical logs for performance and debugging.

Your data is used to:

  • Create and manage your user account (dual profile: artisan/client).
  • Filter and display artisans or work requests according to the selected profession and city.
  • Facilitate direct communication via the built-in chat system (text and audio).
  • Store and synchronize your data securely.
  • Improve the performance and reliability of the application.
  • Display advertisements (banners and interstitials).
